## Описание предметной области
**ROT13** – симметричный шифр подстановки, частный случай шифра Цезаря со сдвигом 13.  
Применяется к латинскому алфавиту (A–Z, a–z).  
Ключевое свойство: самодвойственность — ROT13(ROT13(x)) = x.  
Используется для скрытия текста, не является криптографически стойким.

## Средства отладки, применённые в Visual Studio
- **Точки останова (breakpoints)** – для анализа состояния переменных в методе `Transform`.
- **Пошаговое выполнение (Step Into / Step Over)** – отслеживание каждой итерации цикла.
- **Окна «Локальные» и «Авто»** – просмотр значений `c`, `buffer[i]`.
- **Окно Immediate** – быстрое вычисление `Rot13Cipher.Encrypt("Test")`.
- **Exception Settings** – останов при `ArgumentNullException` для проверки корректности.
- **Диагностические инструменты** – контроль времени выполнения и потребления памяти.
